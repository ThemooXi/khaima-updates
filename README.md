# خيمة — للقصائد الحسينية

تطبيق أندرويد للاستماع إلى القصائد واللطميات الحسينية، مع مزامنة المحتوى بين الأجهزة، ولوحة إدارة للمطور والأدمن.

**المطور:** ميثم عادل (`ThemooXi`)  
**الإصدار الحالي:** 1.4 (versionCode 5)  
**معرف الحزمة:** `com.themoo.khaima`

---

## المميزات

- تصفح وتشغيل القصائد الحسينية (محرم، أربعين، مواليد، لطميات، فضائل…)
- تشغيل من يوتيوب مع استخراج الصوت عبر NewPipe
- مشغّل كامل + شريط تشغيل مصغّر + إشعار الوسائط
- المفضلة والتحميل للاستماع دون اتصال
- حسابات مستخدمين (تسجيل / دخول) مع الضيوف
- تحديثات تلقائية للتطبيق عبر إشعارات النظام
- إشعارات عند إضافة قصائد جديدة
- لوحة أدمن / مطور:
  - رفع قصائد من يوتيوب أو يدوياً
  - تعديل وحذف القصائد (صلاحيات حسب الرتب)

## التقنيات

| التقنية | الاستخدام |
|---------|-----------|
| Kotlin | لغة التطبيق |
| Jetpack Compose | واجهة المستخدم |
| Media3 (ExoPlayer) | التشغيل |
| NewPipe Extractor | استخراج صوت يوتيوب |
| WorkManager | فحص التحديثات بالخلفية |
| JSONBlob | مزامنة القصائد / الحسابات / الرتب |

- **minSdk:** 24  
- **targetSdk / compileSdk:** 35

## البناء والتشغيل

### المتطلبات

- Android Studio Ladybug أو أحدث
- JDK 11+
- اتصال إنترنت (للمزامنة والتشغيل من يوتيوب)

### خطوات التشغيل

```bash
git clone https://github.com/ThemooXi/khaima-app.git
cd khaima-app
```

1. افتح المشروع في Android Studio.
2. انتظر مزامنة Gradle.
3. شغّل على جهاز أو محاكي (`Run`).

لبناء APK إصدار:

```bash
./gradlew :app:assembleRelease
```

الملف الناتج عادةً في:

`app/build/outputs/apk/release/`

├── update/                  # التحديثات والإشعارات
└── ui/screens/              # الشاشات (Home, Player, Admin…)

## التحديثات عن بُعد

ملف التحقق من الإصدار:

`https://raw.githubusercontent.com/ThemooXi/khaima-updates/main/update.json`

عند إصدار نسخة جديدة:

1. ارفع الـ APK على استضافة مباشرة.
2. حدّث `updates/update.json` في مستودع التحديثات.
3. زد `versionCode` و `versionName` في `app/build.gradle.kts`.

## ملاحظات مهمة

- روابط يوتيوب المؤقتة (`googlevideo`) لا تُحفظ؛ يُحفظ رابط المشاهدة الدائم فقط.
- قد يظهر خطأ **HTTP 429** عند كثرة طلبات يوتيوب؛ التطبيق يخزّن الروابط مؤقتاً ويؤخر الطلبات لتقليل ذلك.
- على أندرويد 13+ يجب منح إذن الإشعارات لإظهار تحديثات التطبيق والقصائد الجديدة في شريط الإشعارات.
